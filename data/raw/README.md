# This file contains documentation about the raw data used in the project.

## Raw Data Overview

The raw data for the "Analisis Segmentasi Konten Hoaks di Indonesia" project consists of unprocessed information collected from various sources. This data serves as the foundation for the analysis and insights generated in the dashboard.

## Data Sources

- Social media platforms
- News websites
- User-generated content

## Data Structure

The raw data is organized in a structured format, typically including the following fields:

- **id**: Unique identifier for each entry
- **content**: The text of the hoax content
- **source**: The source from which the content was obtained
- **date**: The date when the content was published or shared
- **category**: The category of the hoax (e.g., political, health, etc.)

## Usage

This raw data will be processed and analyzed to derive meaningful insights regarding hoax content segmentation and trends in Indonesia. It is crucial to maintain the integrity of this data for accurate analysis.
