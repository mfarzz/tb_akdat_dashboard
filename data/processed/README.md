# Processed Data Information

This directory contains processed data that has been cleaned and transformed for analysis. The processed data is structured to facilitate efficient querying and visualization in the dashboard.

## Data Structure

The processed data includes the following key components:

- **Content Segmentation**: Data is categorized based on various segments such as type, source, and credibility.
- **Trends Over Time**: Time-series data reflecting the frequency and nature of hoax content over specified periods.
- **Metadata**: Additional information related to each data entry, including timestamps, sources, and categorization details.

## Usage

The processed data is utilized in the Streamlit application to provide insights and visualizations on hoax content segmentation and trends in Indonesia. Ensure that any modifications to this data maintain the integrity of the structure for accurate analysis.
